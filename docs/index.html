Typography

There are 3 variable fonts available in blossom.

  • Text: $font-family-text - (default: Open Sans)
  • Title: $font-family-title - (default: Raleway)
  • Code: $font-family-code - (default: Source Code Pro)

The default fonts provide a great reading experience and a wonderful "fit all" design.
